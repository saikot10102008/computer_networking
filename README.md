## Main Topics Covered in the Course

The course provides comprehensive coverage of computer networking fundamentals, including:

- Course introduction, methodology, prerequisites, and target audience
- Networking basics: definition of computer networks, data communication components, effectiveness metrics, transmission modes (simplex, half-duplex, full-duplex)
- Network topologies (mesh, star, bus, etc.)
- OSI model and layered architecture
- IP addressing: binary/octet conversions, IPv4 logical addressing, classful vs classless addressing (CIDR), Class A/B/C/D/E details, loopback addresses
- Subnetting: borrowing bits, subnet mask design, variable-length subnet masking (VLSM), supernetting, routing tables & CIDR blocks
- Error control: single-bit vs burst errors, Hamming distance & correction, parity methods, cyclic redundancy check (CRC), checksum
- Flow control & ARQ protocols: transmission/propagation delays, Stop-and-Wait, sliding window, Go-Back-N, Selective Repeat, efficiency & throughput calculations
- Packet headers: IPv4 header (TOS, TTL, fragmentation), TCP header (sequence/acknowledgment numbers, flags, 3-way handshake), TCP congestion control, TCP vs UDP, timers
- Media access control: Aloha, CSMA, CSMA/CD, polling, token passing
- Routing & switching: distance-vector vs link-state routing, circuit vs packet switching
- Application layer protocols: SMTP, POP3, IMAP, DNS hierarchy & queries, FTP, HTTP, ARP, ICMP
- Full OSI model summary and course conclusion

Full details and chapter navigation are available in the YouTube video description.
